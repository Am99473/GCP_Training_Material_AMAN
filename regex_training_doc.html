2Ô∏è‚É£ The re Module in Python

Key Functions:

Function Description Example
re.search() Finds first occurrence re.search('a', 'banana')
re.match() Checks only at the beginning re.match('ban', 'banana')
re.findall() Returns all matches as a list re.findall('a', 'banana')
re.split() Splits string by pattern re.split('\s', 'I like apples')
re.sub() Substitutes pattern with text re.sub('apple', 'mango', 'apple pie')

3Ô∏è‚É£ Basic Regex Patterns and Metacharacters

Symbol Meaning Example Output
. Any single character re.findall("a.", "cat bat mat") ['at', 'at', 'at']
^ Start of string re.findall("^Hi", "Hi there") ['Hi']
$ End of string re.findall("bye$", "goodbye") ['bye']
* 0 or more occurrences re.findall("go*d", "gd good goood") ['gd', 'good', 'goood']
+ 1 or more occurrences re.findall("go+d", "gd good goood") ['good', 'goood']
? 0 or 1 occurrence re.findall("go?d", "gd good") ['gd']
{n} Exactly n times re.findall("a{2}", "aa aaaa a") ['aa', 'aa']
{n,m} Between n and m times re.findall("a{2,3}", "aaaa") ['aaa']
[] Character set re.findall("[aeiou]", "hello") ['e', 'o']
[^] Negation re.findall("[^aeiou]", "hello") ['h', 'l', 'l']
| OR condition re.findall("cat|dog", "I have a cat") ['cat']

4Ô∏è‚É£ Character Classes / Shorthand

Pattern Meaning Example Output
\d Digit re.findall("\d", "A1B2") ['1', '2']
\D Non-digit re.findall("\D", "A1B2") ['A', 'B']
\w Word character (A-Z, a-z, 0-9, _) re.findall("\w", "Hi_123") ['H','i','_','1','2','3']
\W Non-word re.findall("\W", "Hi_123@!") ['@', '!']
\s Whitespace re.findall("\s", "Hi there") [' ']
\S Non-whitespace re.findall("\S", "Hi there") ['H','i','t','h','e','r','e']

5Ô∏è‚É£ Grouping and Capturing

Concept: Use parentheses () to group parts of a pattern.

Example:

text = "Name: Aman, Age: 30" result = re.findall(r"Name: (\w+), Age: (\d+)", text) print(result) # [('Aman', '30')]
Use Case: Extract multiple fields like name, date, or price from structured text.

6Ô∏è‚É£ Anchors and Boundaries

Pattern Meaning Example Output
\b Word boundary re.findall(r"\bcat\b", "cat scatter catalog") ['cat']
\B Non-word boundary re.findall(r"\Bcat\B", "educate") ['cat']

7Ô∏è‚É£ Flags / Modifiers

Used inside re functions to change matching behavior.

Flag Meaning Example
re.IGNORECASE (re.I) Case insensitive re.findall('python', 'PYTHON', re.I)
re.MULTILINE (re.M) Makes ^ and $ match each line re.findall('^Hello', text, re.M)
re.DOTALL (re.S) Makes . match newline too re.search('a.b', 'a\nb', re.S)
